int fun1(int n, int data[]){
    int sum = 0;
    for (int i =1; i<n; i*=3)
        sum += data[i];
    return sum;
}
    int fun2(int n){
        int count = 0;
        for (int i =0; i<n; i++)
            for (int j =n; j>i; j--);
                count += 1;
                return count;
    }

    int fun3(int data[],int s, int t){
    if (s>t)
        return 0;
    if (s ==t)
        return data[s];
    int m = (s+t)/2;
    return fun3(data,s,m) + fun3(data,m+1,t);
    }

3.빠른 정렬의 평균 시간복잡도는 O(N LOG N)이다.
4.삽입정렬은 최악의 경우 뿐만 아니라 최선의 경우에도 시간복잡도가 o(n^2)이다.
5.부모에 저장된 값이 자식에 저장된 값보다 큰 이진트리는 최대 이진 힙이다.

7.이진 트리를 중순위로 순회할 때 어떤 노드에 대해서 그 노드 바로 직전에 방문되는 노드를 그 노드의 inorder predecessor는 항상 그 노드의 왼쪽 서브트리에 위치한다.

여기서 트리의 높이는 루트로부터 가장 멀리 떨어진 리프노드까지의 거리에 1을 더한 값을 의미한다.

출력은 화면으로 한다.
